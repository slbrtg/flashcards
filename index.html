JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Containers for storing data values.

Functions

Blocks of code designed to perform a particular task.

Methods

Actions that can be performed on objects. Properties containing a function definition, property and value.

Arguments

Data passed to the parameters of a function.

Parameters

Arguments used to define a function.

Strings

Primitive values, created from literals.
